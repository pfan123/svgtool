# svgtool

svgtool工具是一款快速合并svg，生成svg symbols的工具。

## 功能特性：

1.支持拖拽文件夹，自动遍历svg文件合并生成symbol文件。

2.对文件viewBox大小不为整进行提示，可以选择自动修正模式系统自动放大处理。

3.导出svg-symbol.svg、svg-symbol.js，可直接引入svg-symbol.js方便使用。
